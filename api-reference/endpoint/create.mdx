---
title: "Get Access Token"
api: "POST https://sandbox.api.scratchie.com/token"
description: "To obtain an access token for the Scratchie open API, your application will need to send a request to the token endpoint, including your client_id and client_secret. The Scratchie open API will then verify your credentials and return an access token that your application can use to make authorized API requests."
---

### Body

<ParamField body="grant_type" type="String">
  The type of grant. Use "client_credentials".
</ParamField>

<ParamField body="client_id" type="String">
  This value is supplied to you by Scratchie open API.
</ParamField>

<ParamField body="client_secret" type="String">
  This value is supplied to you by Scratchie open API.
</ParamField>

### Response

<ResponseField name="access_token" type="String">
  The access token received upon successful authorization.
</ResponseField>

<ResponseField name="expires_in" type="Number">
  The expiration time of the access token in seconds (e.g., 3600 seconds).
</ResponseField>

<ResponseField name="token_type" type="String">
  The type of token, typically "Bearer".
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://{BASE_URL}/token' \
--header 'Content-Type: application/json' \
--data-raw '{
    "grant_type": "client_credentials",
    "client_id": "IZRYGADXUZ5QSCSISSZ3FBMN91CSC3T23YDOXT",
    "client_secret": "tyyCoslcmCaLfsdI0jExqWKzl0oK6d4OcTWu5gI4trj"
}'
```
</RequestExample>

</ResponseExample>
