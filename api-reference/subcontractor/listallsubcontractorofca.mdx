---
title: "Show Subcontractor List Of CA"
api: "GET https://sandbox.api.scratchie.com/ca/subcontractor-list/:caId"
description: "The Get Subcontractor List Of CA API allows you to retrieve detailed information about a specific user within Scratchie."
---

### Request Parameters

<ParamField body="caId" type="Number">
  This field represents the [caId](https://{BASE_URL}/ca/create) endpoint for obtaining the ID.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Array">
  List of subcontractors for the specified CA.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```http
GET https://{BASE_URL}/ca/subcontractor-list/355
Authorization: Bearer <access_token>
```
