---
title: "Show Subcontractor"
api: "GET https://sandbox.api.scratchie.com/subcontractor/subcontractor-detail/:subcontractorId"
description: "The Show Subcontractor API allows you to retrieve detailed information about a specific subcontractor."
---

### Request Parameters

<ParamField body="subcontractorId" type="Number">
  This field represents the [subcontractorId](https://{BASE_URL}/subcontractor/create) endpoint for obtaining the Id.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  Detailed information about the subcontractor.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```http
GET https://{BASE_URL}/subcontractor/subcontractor-detail/285
Authorization: Bearer <access_token>
```

</RequestExample>
