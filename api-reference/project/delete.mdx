---
title: "Delete Project"
api: "DELETE https://sandbox.api.scratchie.com/project/view/:projectId"
description: "The Delete Project API enables you to remove a specific project from the Scratchie Open API system."
---

### Path Parameters

<ParamField body="projectId" type="Number">
  This field represents the project ID for the project you want to delete.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  An empty object as the response data.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```bash
curl --location --request DELETE 'https://{BASE_URL}/project/view/346782' \
--header 'Authorization: Bearer <access_token>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "status": 200,
  "err": 0,
  "data": {},
  "msg": "Project successfully deleted"
}
```

</ResponseExample>
