---
title: "Create Project"
api: "POST https://sandbox.api.scratchie.com/project/create"
description: "The Create Project API allows you to create a new project in Scratchie. Projects are a fundamental component of Scratchie and provide a structured way to organize worker management."
---

### Body

<ParamField body="Name" type="String">
  This field denotes the official name assigned to your project.
</ParamField>

<ParamField body="BuilderProjectReferenceNumber" type="String">
  This is the unique reference number assigned to the project by the builder.
</ParamField>

<ParamField body="Address" type="String">
  This field refers to the address of your project's physical location.
</ParamField>

<ParamField body="minimumAmount" type="Number">
  The minimumAmount parameter represents the required wallet size for your project, which must exceed $2,000.
</ParamField>

### Headers

<ParamField body="Content-Type" type="String">
  application/json
</ParamField>

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>
