---
title: "Award in Participants API"
api: "POST https://sandbox.api.scratchie.com/award/contributor-in-award"
description: "The Join Award API for Workers empowers workers to participate in previously created awards."
---

### Request Parameters

<ParamField body="WorkerId" type="Number">
  This field represents the [worker ID](https://{BASE_URL}/worker/view/:workerId) endpoint for obtaining the ID.
</ParamField>

<ParamField body="AwardUniqueNo" type="String">
  This field represents the [Award ID](https://{BASE_URL}/award/award-detail/:AwardId) endpoint for obtaining the ID.
</ParamField>

### Headers


<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  Details about the contribution to the award.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```json
POST https://{BASE_URL}/award/contributor-in-award
Content-Type: application/json
Authorization: Bearer <access_token>

  "WorkerId": 1254,
```
