---
title: "Show Award Detail"
api: "GET https://sandbox.api.scratchie.com/award/award-detail/:AwardId"
description: "The Get Award Details API allows you to retrieve detailed information about a specific award."
---

### Request Parameters

<ParamField body="AwardId" type="Number">
  This field represents the [Award ID](https://{BASE_URL}/award/create) endpoint for obtaining the ID.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Array">
  Detailed information about the award.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```json
GET https://{BASE_URL}/award/award-detail/3582
Authorization: Bearer <access_token>
```
