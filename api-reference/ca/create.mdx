---
title: "Create CA"
api: "POST https://sandbox.api.scratchie.com/ca/create"
description: "The Create CA API allows you to seamlessly add a new Contracts Administrator (CA) to Scratchie and associate them with a project."
---

### Request Parameters

<ParamField body="firstName" type="String">
  First name of the CA.
</ParamField>

<ParamField body="lastName" type="String">
  Last name of the CA.
</ParamField>

<ParamField body="email" type="String">
  Email address of the CA.
</ParamField>

<ParamField body="projectId" type="Number">
  ID of the project associated with the CA.
</ParamField>

<ParamField body="password" type="String">
  Password for the CA.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  Details of the newly created CA.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```json

Content-Type: application/json
Authorization: Bearer <access_token>
```
