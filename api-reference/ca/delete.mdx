---
title: "Delete CA"
api: "DELETE https://sandbox.api.scratchie.com/ca/delete/:caId"
description: "The Delete CA API enables you to remove a specific user from Scratchie."
---

### Request Parameters

<ParamField body="caId" type="Number">
  This field represents the https://{BASE_URL}/ca/create endpoint for obtaining the ID.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  An empty object.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```http
DELETE https://{BASE_URL}/ca/delete/14
Authorization: Bearer <access_token>
```
</RequestExample>

<ResponseExample>

```json Response
{
  "status": 200,
  "err": 0,
  "data": {},
  "msg": "CA successfully deleted."
}
```
</ResponseExample>
