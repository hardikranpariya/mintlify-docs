---
title: "Join Another Project API for Workers"
api: "POST https://sandbox.api.scratchie.com/award/multipleproject-contributor-add"
description: "The Join Another Project API for Workers enables workers to join a different project in the same company."
---

### Request Parameters

<ParamField body="WorkerId" type="Number">
  This field represents the ID of the worker who wants to join another project, obtained from the worker's details.
</ParamField>

<ParamField body="SubContractorId" type="Number">
  This field represents the ID of the subcontractor or the new project the worker wants to join, obtained from the subcontractor details.
</ParamField>

### Headers

<ParamField body="Content-Type" type="String">
  application/json
</ParamField>

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  An empty object.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

```json
POST https://{BASE_URL}/award/multipleproject-contributor-add
Content-Type: application/json

{
  "SubContractorId": 283,
  "WorkerId": 1257

```
