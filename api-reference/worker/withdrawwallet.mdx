---
title: "Withdraw From User Scratchie Wallet"
api: "POST https://sandbox.api.scratchie.com/worker/withdraw-subwallet"
description: "The Withdraw from User Scratchie Wallet API allows you to initiate a fund withdrawal from a user's Scratchie wallet."
---

### Request Parameters

<ParamField body="workerId" type="Number">
  This field represents the ID of the worker you want to withdraw funds for, obtained from the previous worker creation endpoint.
</ParamField>

<ParamField body="amount" type="Number">
  The amount to withdraw from the user's Scratchie wallet.
</ParamField>

<ParamField body="Type" type="String">
  Only expect 'amount'.
</ParamField>

### Headers

<ParamField body="Authorization" type="String">
  Bearer `<access_token>`
</ParamField>

### Response

<ResponseField name="status" type="Number">
  The HTTP status code of the response.
</ResponseField>

<ResponseField name="err" type="Number">
  An error code or indicator (0 if successful).
</ResponseField>

<ResponseField name="data" type="Object">
  An empty object.
</ResponseField>

<ResponseField name="msg" type="String">
  A message indicating the result of the operation.
</ResponseField>

<RequestExample>

```json
POST https://{BASE_URL}/worker/withdraw-subwallet
Content-Type: application/json
Authorization: Bearer <access_token>

{
  "workerId": 1257,
  "amount": 123,
  "Type": "amount"
}
```

</RequestExample>

<ResponseExample>

```json Response
{
  "status": 200,
  "err": 0,
  "data": {},
  "msg": "Success! Scratchie wallet balance has been transferred to your bank account."
}
```

</ResponseExample>
